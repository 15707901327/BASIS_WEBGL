着色器概述

GLSE ES概述

GLSE ES编程语言是在OpenGL着色器语言（GLSL）的基础上，删除和简化一部分功能后形成的。

编码规范
  • 程序是大小写敏感的
  • 每个语句都应该以一个英文分号(;)结束
注释
  • 单行注释 //
  • 多行注释 /**/

限定字

包含存 存储限定字（const、attribute、uniform、varying）和 精度限定字（precision） 两种。

着色器中的变量

变量概述

声明在函数外部的是全局变量，声明在函数内部的是局部变量。

规范
  • 只包含a-z\A-Z和下划线_
  • 首字母大写
  • 不能使用保留字

片元着色器中的内置变量

片元着色器中的内置变量
类型和变量名 描述 实例
vec4 gl_FragColor 指定片元颜色（RGBA格式）
vec4 gl_FragCoord 片元的窗口坐标，第1个x和第2个y分量表示在canvas坐标系统中（窗口坐标系统）的坐标值。z值是深度值。
vec4 gl_PointCoord 片元在被绘制的点内的坐标（0.0到1.0）
  • 实现画圆

顶点着色器中的内置变量

顶点着色器的内置变量
类型和变量名 描述
vec4 gl_Position 表示顶点的位置，注意变量必须被赋值，否则着色器就无法正常工作
float gl_PointSize 表示点的尺寸(像素数)， 不是必须的，不过不赋值，着色器会取默认值1.0

数据值类型

类型 描述
数值类型 支持整数和浮点数
布尔值类型 true\false
