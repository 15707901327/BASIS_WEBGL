ctx.fillStyle

ctx.fillRect()

ctx.createRadialGradient()

ctx.fillStyle = 'rgba(0,0,255,0.5)';

设置填充颜色为蓝色

参数：

rgb表示颜色（0-255）a ： 透明度 （0.0-1.0）

实例
  • 绘制蓝色矩形

ctx.fillRect(120,10,150,150);

使用填充颜色填充矩形

参数：

前两个指定了待绘制矩形的顶点在《canvas》中的坐标，后两个指定了矩形的宽度和高度

实例
  • 绘制蓝色矩形

ctx.createRadialGradient(x0,y0,r0,x1,y1,r1);

方法创建放射状/圆形渐变对象

参数：
  • x0:渐变的开始圆的 x 坐标
  • y0:渐变的开始圆的 y 坐标
  • r0:开始圆的半径
  • x1:渐变的结束圆的 x 坐标
  • y1:渐变的结束圆的 y 坐标
  • r1:结束圆的半径
