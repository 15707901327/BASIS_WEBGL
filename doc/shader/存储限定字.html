attribute 变量

说明
  • attribute变量只能出现在顶点着色器中，只能被声明为全局变量，被用来表示逐顶点的信息。
声明变量

变量的声明格式：存储限定符->类型->变量名

attribute vec4 a_Position;

  • 关键词attribute被称为存储限定符,它表示接下来的变量是一个attribute变量
作用
  • 被用来从外部向顶点着色器内传输数据
实例一：在顶点着色器中声明变量，并使用 
        // 顶点着色器程序
        var VSHADER_SOURCE =
          'attribute vec4 a_Position;\n' +
          'void main() {\n' +
          '  gl_Position = a_Position;\n' + //设置坐标
          '  gl_PointSize = 10.0;\n' + //设置尺寸
          '}\n';

uniform 变量

说明
  • uniform 变量可以用在顶点着色器和片元着色器中，且必须是全局变量。
  • uniform 变量是只读的，它可以是除了数组或结构体之外的任意类型。
  • 如果顶点着色器和片元着色器中声明了同名的uniform 变量，那么它就会被这两种着色器共享。
  • uniform 变量传输的是那些对于所有顶点都相同（或与顶点无关的数据）
声明变量

变量的声明格式：存储限定符->类型->变量名

uniform vec4 u_FragColor

实例一：在片元着色器中声明变量并使用 
        // 片元着色器
        var FSHADER_SOURCE =
        'uniform vec4 u_FragColor;\n' +
        'void main() {\n' +
        '  gl_FragColor = u_FragColor;\n' +
        '}\n';

varying 变量

说明
  • varying 变量必须是全局变量，它的任务是从顶点着色器向片元着色器传输数据，我们必须在两种着色器中声明同名，同类型的变量。
  • 顶点着色器中赋给varying变量的值并不是直接传递给了片元着色器的varying变量，这其中发生了光栅化的过程。
声明

变量的声明格式：存储限定符->类型->变量名

varying vec4 v_Color
