重复执行函数方法

  • setInterval()
  • requestAnimationFrame()
  • cancelAnimationFrame()

setInterval(func,delay)

每隔delay时间间隔，调用func函数

参数
  • func:指定需要多次调用的函数
  • delay：指定时间间隔（以毫秒为单位）
返回值

Time id

说明
  • 不管标签页是否被激活，其中的setInterval（）函数都会反复调用func，如果标签页较多，就会增加浏览器的负荷

requestAnimationFrame(func)

请求浏览器在将来某个时刻回调函数func以完成重绘。我们应当在回调函数最后再次发起该请求

参数：
  • func:指定将来某时刻调用的函数。函数将会节后一个time参数，用来表明此次调用的时间戳
返回值

Request id

说明：
  • 避免在未激活的标签页上运行动画。
  • 无法指定重复调用的间隔；函数func会在浏览器需要网页的某个元素重绘时被调用。
  • 调用一次执行一次

cancelAnimationFrame(requestID)

取消由requestAnimationFrame发起的请求

参数
  • requestID:指定requestAnimationFrame()的返回值
返回值： 无
