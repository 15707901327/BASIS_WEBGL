ANGLE_instanced_arrays

EXT_texture_filter_anisotropic

各向异性过滤（AF）

当以倾斜角度查看纹理图元时，AF可提高mipmap纹理访问的质量。仅使用mipmapping，这些查找具有平均灰色的趋势。

OES_texture_float

OES_texture_float_linear

OES_texture_half_float

判断是否支持支持`gl.HALF_FLOAT_OES`作为纹理类型

OES_vertex_array_object(顶点数组扩展对象)

扩展方法：
  • ext.createVertexArrayOES() 创建一个新的 WebGLVertexArrayObject
  • ext.deleteVertexArrayOES() 删除一个给定的 WebGLVertexArrayObject
  • ext.isVertexArrayOES() 如果参数是一个 WebGLVertexArrayObject则返回 true
  • ext.bindVertexArrayOES() 绑定一个给定的WebGLVertexArrayObject到缓冲区。
