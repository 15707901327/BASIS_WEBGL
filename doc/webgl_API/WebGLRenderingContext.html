texUnit

指定准备激活的纹理单元,其值是 gl.TEXTUREI ，其中的 I 在 0 到gl.MAX_COMBINED_TEXTURE_IMAGE_UNITS - 1范围内。例如：gl.TEXTURE0、gl.TEXTURE1......gl.TEXTURE7。最后的数字表示纹理单元的编号。

WebGLRenderingContext.js, line 604

指定了一个将传入像素深度与当前深度缓冲区值进行比较的函数

Name Type Description
func GLenum

用来设置视口，即指定从标准设备到窗口坐标的 x、y 仿射变换。
设置gl.drawArrays()和gl.drawElements()函数的绘图区域。在canvas上绘图时，x和y就是canvas中的坐标

Name Type Description
x GLint

用来设定视口的左下角水平坐标。默认值：0。
y GLint

用来设定视口的左下角垂直坐标。默认值：0。
width GLsizei

用来设定视口的宽度。默认值：canvas 的宽度。
height GLsizei

用来设定视口的高度。默认值：canvas 的高度。
